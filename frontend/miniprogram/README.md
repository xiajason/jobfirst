# ADIRP数智招聘小程序

基于原型设计文件打造的智能求职小程序，为用户提供便捷的求职体验。

## 项目概述

ADIRP数智招聘小程序是一个现代化的求职平台，集成了职位搜索、简历管理、在线聊天、投递记录等功能，为用户提供全方位的求职服务。

## 功能特性

### 🏠 首页
- 智能搜索：支持职位、公司、技能关键词搜索
- 轮播推荐：展示热门活动和推荐内容
- 快捷功能：推荐职位、附近职位、简历管理、投递记录
- 热门职位：展示当前热门招聘职位
- 推荐公司：展示优质企业信息
- 行业分类：按行业分类浏览职位

### 💼 职位列表
- 多维度筛选：地点、薪资、经验等筛选条件
- 智能排序：推荐、最新、附近等多种排序方式
- 职位详情：完整的职位信息展示
- 一键投递：快速投递简历
- 收藏功能：收藏感兴趣的职位

### 💬 聊天功能
- 实时沟通：与HR进行实时对话
- 消息管理：未读消息提醒、消息历史
- 在线状态：显示HR在线状态
- 筛选功能：按未读、在线状态筛选聊天

### 👤 个人中心
- 用户信息：个人资料管理
- 数据统计：投递记录、面试邀请、收藏职位等统计
- 简历管理：创建、编辑、管理个人简历
- 投递记录：查看投递历史和状态
- 面试邀请：管理面试通知
- 应用设置：通知、隐私等设置

## 技术架构

### 前端技术栈
- **框架**：微信小程序原生开发
- **样式**：WXSS + 响应式设计
- **状态管理**：小程序全局数据管理
- **网络请求**：wx.request API
- **存储**：本地存储 + 云存储


## 开发环境

### 环境要求
- 微信开发者工具 1.06.2307260 或更高版本
- Node.js 16.0 或更高版本
- 微信小程序 AppID

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd miniprogram
   ```

2. **打开微信开发者工具**
   - 导入项目
   - 选择小程序项目
   - 填入 AppID

3. **配置项目**
   - 修改 `project.config.json` 中的 AppID
   - 配置后端API地址（在 `app.js` 中）

4. **运行项目**
   - 点击编译
   - 在模拟器中预览

## 配置说明

### 后端API配置
在 `app.js` 中配置后端服务地址：
```javascript
globalData: {
  baseUrl: 'http://localhost:8080', // 修改为实际的后端地址
  // ...
}
```

### 小程序配置
在 `app.json` 中配置页面路由、窗口样式、底部导航等：
```json
{
  "pages": [
    "pages/index/index",
    "pages/jobs/jobs",
    "pages/chat/chat",
    "pages/profile/profile"
  ],
  "window": {
    "navigationBarTitleText": "ADIRP数智招聘"
  }
}
```

## 页面说明

### 首页 (pages/index/index)
- 搜索功能：支持关键词搜索
- 轮播图：展示活动信息
- 快捷入口：常用功能快速访问
- 推荐内容：热门职位和公司推荐

### 职位列表 (pages/jobs/jobs)
- 筛选器：多维度筛选条件
- 职位卡片：展示职位基本信息
- 操作按钮：收藏、投递等功能
- 分页加载：支持上拉加载更多

### 聊天列表 (pages/chat/chat)
- 聊天记录：显示所有聊天会话
- 未读提醒：未读消息数量显示
- 在线状态：HR在线状态指示
- 筛选功能：按条件筛选聊天

### 个人中心 (pages/profile/profile)
- 用户信息：头像、姓名、简介
- 数据统计：各项数据汇总
- 功能菜单：分类展示所有功能
- 设置选项：应用相关设置

## 开发规范

### 代码规范
- 使用 ES6+ 语法
- 遵循微信小程序开发规范
- 保持代码简洁和可读性
